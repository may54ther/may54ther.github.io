## 주요 시스템 파일 경로
```
| 파일 경로          | 설명				|
|-------------------|-----------------------------------|
| /etc/passwd       | 사용자 계정 기본 정보		|
| /etc/shadow       | 사용자 암호 정보 (암호화 저장)	|
| /etc/group        | 그룹 정보				|
| /etc/gshadow      | 그룹 암호 정보			|
| /etc/sudoers      | sudo 권한 사용자 정의		|
```

## 사용자 관리

### 사용자 추가
```sh
# 홈 디렉터리 X
useradd [사용자]

# 홈 디렉터리 O
adduser [사용자]
```
### 사용자 삭제
```sh
# 계정 삭제
deluser [사용자]

# 계정 + 홈 디렉터리 삭제
deluser [사용자] --remove-home

# 계정 + 홈 + 메일 스풀 삭제
deluser [사용자] --remove-all-files
```
### 사용자 수정
```sh
# 로그인 셸 변경
usermod -s /bin/bash [사용자]

# 홈 디렉터리 변경 (기존 디렉터리 이동 포함)
usermod -d /home/newhome -m [사용자]

# UID 변경
usermod -u 1234 [사용자]

# 잠금 / 잠금 해제
usermod -L [사용자]   # 계정 잠금
usermod -U [사용자]   # 계정 잠금 해제
```
### 사용자 비밀번호 설정
```sh
# 비밀번호 설정
passwd [사용자]

# 비밀번호 제거 (빈 암호 허용)
passwd -d [사용자]

# 만료 설정 (다음 로그인 시 변경 요구)
passwd -e [사용자]
```
### 사용자 그룹 설정
```sh
# 사용자 그룹 추가 (보조 그룹)
usermod -aG [그룹] [사용자]

# 그룹에서 사용자 제거
gpasswd -d [사용자] [그룹]

# 초기 그룹 설정
usermod -g [그룹] [사용자]

# 현재 그룹 확인
id [사용자]
groups [사용자]
```
###  사용자 sudo 권한 설정
```sh
# 사용자에게 sudo 권한 부여
usermod -aG sudo [사용자]
```

## 그룹 관리

### 그룹 생성
```sh
groupadd [그룹명]
```
### 그룹 정보 확인
```sh
getent group [그룹명]
```
### 그룹 삭제
```sh
groupdel [그룹명]
```
